Linux下快速浏览粘贴表情的解决方案

2019-08-20
Linux

Contents

长期以来，Linux下楼主只能按一般方式用file manager查找和插入图片的方式，由于 表情一般很多而导致查找非常不便。一不留神，终于让楼主发现了一个好的解决方法。

适用人群：像楼主这样特别喜欢用表情包的，不发表情不舒服斯基

适用系统: Linux （Mac下可以Alfred简单解决）

基本工具: sxiv 和 xclip (以及一个支持快捷键绑定的window manager或graphical environment)

解决方法: 思路就是用快捷键调用sxiv以thumbnail mode打开表情包所在的文件夹，浏览选中要使用的表情后快捷键拷入剪贴板中，在网页中C-v就好了。

注意: 由于xclip的问题，目前只支持png，如果想用jpg，可以自己写个python拷图片进clipboard。

细节:

  1. 安装sxiv和xclip，并配置sxiv所需扩展，即 cp /usr/share/sxiv/exec ~/.config/sxiv/ (如果是用Archlinux。如不是，就自己找找安装目录）

  2. 绑定一个全局快捷键，如我用super+m，执行如下命令（你可以搞个script，我用的wm i3所以绑键很方便不需要） sxiv -t ~/Downloads/acfun_emoji/ (请替换这个地址为你的表情包目录）

  3. 使用时，用super+m打开sxiv浏览表情（用h,i,j,k随你浪游），如下图；选择好要的表情后，回车打开，然后按C-x C-v拷入剪贴板，按q关掉sxiv的临时窗口；切回网页粘贴即可。
